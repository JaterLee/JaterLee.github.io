# 需求文档：生活小记

## 背景

新增一个「生活小记」栏目，作为个人日常记录、感悟、随笔的集合。类似五代十国史模块但内容不是历史，而是个人生活相关的短篇文字。

## 功能列表

- [P0] 在左侧导航栏新增「生活小记」卡片入口
- [P0] 点击后右侧展示生活小记内容区
- [P0] 内容按日期倒序排列
- [P1] 与现有模块风格一致（双面板布局）
- [P1] 支持内容的 Markdown 渲染
- [P2] 图片/多媒体嵌入支持

## 验收标准

- [ ] 导航栏出现「生活小记」卡片，带图标
- [ ] 点击切换到生活小记内容
- [ ] 内容正常加载展示
- [ ] 移动端可横向滑动切换到此模块
- [ ] 按日期倒序排列

## 依赖/风险

- 需注册到 `data/modules.json` 和 `module-registry.js`
- 内容存储在 `content/life/` 目录
- 样式需新建 `assets/css/module-life.css`
- 逻辑需新建 `assets/js/module-life.js`
- 在 `index.html` 中新增模块面板
